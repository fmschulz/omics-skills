# Journal Metrics References

This skill ships a curated table of journal-level metrics in `journal_metrics_2024.tsv`.

## Scope

- Metric year: `2024`
- Verification date: `2026-03-19`
- Intended use: add journal-level context to paper-level OpenAlex citation data

## Source policy

- Prefer official publisher pages that publicly display Journal Impact Factor values.
- When no public official page could be located, the row is labeled `third_party`.
- The current table includes broad journals, bio/computer-science journals, and medical journals.

## Important caveat

Clarivate's Journal Impact Factor is a licensed metric. This table contains only values that were publicly visible on the web during skill creation. It is a convenience reference, not a complete substitute for licensed Journal Citation Reports access.

The `Science` row is intentionally labeled `third_party` because no public AAAS page exposing the current JIF was located during skill creation on March 19, 2026.
