# Literature entry points — interdomain HGT

Starting references for framing HGT findings and the analysis playbook. Expand with
`/polars-dovmed` / `/biorxiv-search` for the specific inferred group.

- `2021-irwin-euk-virus-hgt.md` — systematic HGT between eukaryotes and viruses;
  bidirectionality; euk-derived viral genes and host-manipulation functions.
- `2020-green-algae-giantvirus-channelrhodopsin-lgt.md` — direct precedent for
  green-algae <-> giant-virus lateral transfer (anion channelrhodopsins).
- `2024-virus-virus-gene-gain.md` — virus-to-virus transfer is a major source of
  giant-virus gene gain; a key confounder to rule out for "host-derived" calls.

General playbook notes:
- NCLDV/Nucleocytoviricota routinely carry host-derived metabolic/accessory genes;
  recent acquisitions form well-supported viral clades nested within a host lineage.
- Chlorophyte-infecting giant viruses (chloroviruses + prasinoviruses) are a
  well-studied host<->virus exchange system; useful comparison set for green-algal hosts.
