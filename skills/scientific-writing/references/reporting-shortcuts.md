# Reporting Shortcuts

Use these shortcuts when the user does not specify a reporting framework.

## Common guideline mapping

- randomized trial: CONSORT
- observational cohort, case-control, or cross-sectional study: STROBE
- systematic review or meta-analysis: PRISMA
- study protocol: SPIRIT
- diagnostic accuracy study: STARD
- case report: CARE

If the venue gives its own checklist, that overrides the default mapping.

## Citation policy

- Validate DOI metadata before introducing a new reference when practical.
- Prefer trusted bibliography files over ad hoc citation generation.
- Keep citation edits auditable.
- If a citation cannot be verified, mark it for follow-up instead of guessing.

## Minimum final-pass checks

- section claims match available evidence
- numbers in prose match tables or structured summaries
- methods trace to actual artifacts
- citation style matches the venue when known
- unresolved gaps are stated explicitly
