# Supporting Skills

Use supporting skills only when they solve a concrete gap in the current manuscript task.

## Citation and reference work

- `crossref-lookup`: DOI lookup, bibliography audits, and citation metadata cleanup
- `scientific-impact-assessment`: OpenAlex citations, optional Altmetric, and bundled journal metrics when discussing reach or influence
- `research-lookup` or `literature-review`: background evidence gathering when the user explicitly wants new sources

Do not use these tools to pad references. Use them only to verify or fill clearly scoped citation needs.

## Writing and review

- `humanize-scientific-writing`: final-pass prose cleanup after factual accuracy is already locked
- `manuscript-review-council`, `peer-review`, or `proposal-review`: heavier review passes when the user wants formal critique

## Reporting and methods support

- `bio-workflow-methods-docwriter`: methods extraction from workflow artifacts
- `bio-logic` or `scientific-critical-thinking`: claim rigor checks when interpretation is the main risk

If a supporting skill is unavailable in the current environment, continue with the base scientific-writing workflow instead of blocking.
