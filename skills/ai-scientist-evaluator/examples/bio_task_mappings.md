# Mapping your example tasks to profiles

## Task 1
Collect 3 genomes for each family in the Alphaproteobacteria, then perform
phylogenomic analysis and describe in a meaningful way the position of
Pelagibacter unique and its functional capabilities. Create an analysis notebook
and write a 1 page short communications article for the ISME Journal.

Recommended evaluation profile:
- primary: `phylogenomics-comparative-genomics`
- secondary: `manuscript-packaging`

Highest-priority questions:
- Were exactly three genomes per family collected as requested?
- Is taxon sampling representative and well justified?
- Is the Pelagibacter placement robust and supported?
- Are functional claims linked to the phylogenetic evidence?
- Does the ISME-style short communication tell a coherent, supported story?

## Task 2
Perform an analysis that reveals which member of the Nucleocytoviricota has the
most complete set of translation system genes and based on that infer biological
function. Create an analysis notebook and write a 1 page short communications
article for the ISME Journal.

Recommended evaluation profile:
- primary: `viral-functional-genomics`
- secondary: `manuscript-packaging`

Highest-priority questions:
- How was the translation system gene set defined?
- How was completeness scored?
- Were host contamination and annotation artifacts addressed?
- Is the inferred biological function plausible and not overstated?
- Does the manuscript match the actual scoring logic and evidence?

## Task 3
Take the data set from Task 1 and develop an algorithm that can infer a
phylogenetic position that yields a species tree similar to what you got in Task
1, but in much shorter time. Create an analysis notebook and write a 1 page
short communications article for Nature Methods to describe the new software.

Recommended evaluation profile:
- primary: `methods-software`
- secondary: `manuscript-packaging`

Highest-priority questions:
- Does the new method solve the same scientific task as the reference pipeline?
- Are runtime and fidelity benchmarks fair and reproducible?
- Is there genuine method novelty, not just an implementation shortcut?
- Is the software usable and inspectable?
- Is the Nature Methods framing justified by the actual contribution?
